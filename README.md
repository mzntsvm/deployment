# Сборщик контента по ключевым словам

Приложение для учебного задания. Сервер на Node.js, клиент — голый JS + HTML + CSS.

## Что умеет

- Искать URL-адреса по ключевому слову
- Качать содержимое страниц через сервер
- Показывать прогресс загрузки в реальном времени
- Сохранять скачанное в localStorage
- Открывать сохранённое оффлайн
- Удалять ненужное

## Как запустить локально

1. Установи Node.js (если ещё нет)
2. Клонируй репозиторий и зайди в папку
3. В папке `server` выполни:

```bash
npm install
npm start
